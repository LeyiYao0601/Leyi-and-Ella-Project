# Leyi-and-Ella-Project
Project Repository for Longitudinal Data Analysis Practical 25/26

longitudinal modelling of the UCL cohorts COVID-19 survey
data.


02_clean.R: apply agreed cleaning rules and create a saved analysis dataset;
03_descriptive.R: produce tables and figures without fitting hypothesis
models;
04_modelling.R: fit the longitudinal model 
05_diagnostics.R: examine model assumptions and influential observations;
06_sensitivity.R: compare complete-case, missing-data, and alternative
specification results.
